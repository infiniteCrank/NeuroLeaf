Stage 1: AutoComplete

The first ELM model is trained on a rich set of multilingual greetings and suggests completions as the user types. This helps clean up noisy or incomplete input.

Stage 2: Language Classification

The completed text is passed into the second ELM, which classifies the language and returns a confidence score.

üéØ Why Chain ELMs?

  • Noise resilience: Autocomplete reduces typos and helps guide input
  • Intent-awareness: Language classifier acts on semantically refined input
  • Modularity: Swap models independently and debug in isolation

üõ†Ô∏è Technical Details

Both models are powered by the astermind JavaScript library. Training is done using single-pass feedforward inference, random hidden layers, and output weights calculated analytically.

Data is sourced from a multilingual CSV file with 500+ greetings in English, Spanish, and French.
