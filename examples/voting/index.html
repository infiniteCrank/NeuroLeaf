This demo classifies the language of greetings in real-time using chained Extreme Learning Machines (ELMs) with a voting ensemble from multiple models.

üîç Current Architecture:

  1. AutoComplete ELM predicts a complete greeting from the user‚Äôs partial input.
  2. CharacterLangEncoderELM encodes the greeting into a normalized vector.
  3. Metadata includes length, character diversity, vowel and punctuation ratios.
  4. FeatureCombinerELM predicts language based on vector + metadata.
  5. LanguageClassifier separately predicts based on encoded vector only.
  6. VotingClassifierELM combines both predictions and outputs a final decision.

Output indicators:
[C] = Combiner prediction,
[R] = Refiner override,
[AC] = Based on autocomplete suggestion,
[V] = Final ensemble decision via VotingClassifierELM
